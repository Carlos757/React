# Mini proyecto de gestión de veterinaria

Realizado con tecnologías como lo son:

-   Vite
-   React.js
-   Tailwind CSS
-   Font Awesome
-   React-Toastify
-   entre otros.

## Instalación

Posisionarse en la carpeta raíz y ejecutar los siguientes comandos:

```
cd citas-react
npm install
npm run dev
```
